Classical ML Feature Engineering

The ML pipeline applies lag shifting and rolling averages to historical sales data before fitting the XGBoost Regressor. 

# Feature engineering lag variables
df['sales_lag_7'] = df.groupby('product_id')['sales'].shift(7)
df['sales_lag_14'] = df.groupby('product_id')['sales'].shift(14)
df['sales_roll_mean_7'] = df.groupby('product_id')['sales'].shift(1).rolling(window=7).mean()
df['day_of_week'] = df['date'].dt.weekday
Delta Table Blueprints

Delta Lake table storing quantitative metrics write-backs from agent decisions. 

CREATE TABLE hive_metastore.observability.retail_evals (
    audit_id STRING,
    timestamp TIMESTAMP,
    product_id INT,
    xgboost_forecast_sum DOUBLE,
    stockout_probability DOUBLE,
    decision STRING,
    reorder_quantity INT
) USING DELTA;
